## 6주차수업내용
실습1: 자바스크립트 활용하기 -  JS 폴더를 생성하고 파일 복사한다. js에 다운받은 부트5(.js) 라이브러리를 복사하여 내부 로컬에서 연동한다. Html 파일 내에서 직접 js삽입(head)한다.
    <script>
    window.onload = function() {
    alert("메인 페이지 로딩 완료");
    }
    </script>
    를 넣어 페이지 새로고치면 "메인 페이지 로딩 완료"라는 팝업 창이 뜨게 된다. 왜냐하면 window.onload가 윈도우를 열(로드)때마다 뜨게 하는 것이기 때문이다.

실습2:  자바스크립트 활용하기(부트5:외부) - "무료 플레이를 클릭하면 즐거운 플레이 되십시오"라는 화면이 작게 뜨게 된다. 아까와 다른 이유는 이번 코드는 버튼을 클릭하였을때 뜨기로 되어 있기 때문이다. 그래서 새로고침을 한다고 했을때 뜨지 않고 무료 플레이 버튼을 눌러야 뜨는 것이다.

실습3: 자바스크립트 기본 문법1 - 블록 스코프 내에서만 접근이 가능한 if문이다. 첫번째 줄은 접근 가능하고 두번째 줄은 접근이 불가능하다. 세번째 줄은 에러가 나지 않는데 두번째 줄을 주석처리하고 페이지를 다시 로딩한다면 세번째줄도 에러가 나는 것을 볼 수 있다. 이렇게 두번째 줄을 주석을 해야 세번째줄이 에러가 나는 이유는 인터프리터 언어는 한줄씩 실행이 되기 때문이다. 즉, 두번째 줄에서 에러가 났기 때문에 멈춰서 세번째 줄은 실행되지 않은 것이다.

실습4: 자바스크립트 기본 문법2 - var은 고전적인 방식이다. 코드를 봤는데 똑같은 변수를 선언하고 값을 변경했는데 가능한 방법이다. let과 const는 되지 않는다. 호이스팅이라고 하는 것은 코드 실행전, 선언부를 해당 스코프 상단으로 끌어올린다.

실습5: LOL 기능 구현 중 실시간 챔피언 검색하기 - 검색창을 만들어뒀지만 아무런 반응이 없기 때문에 기능을 추가해야한다. 텍스트창 같은 것이 있는 것을 form창이라고 부른다. 이를 자바스크립트를 이용해 버튼을 누르면 실행이 되도록 본다. search를 searchForm으로 바꿔주고 id가 고유값이다. 그리고 search.js 파일을 만들어서 html 최하단에 연동한다. 그러고 페이지에서 검색을 해보면 오류가 난다. 다시 indev=x.html.dp 하단으로 가서  <script src="js/search.js"></script>를 해주면 오류가 나지 않는다. 내가 클릭을 하는 순간 타입 중 submit이 실행이 되는 것이다.
